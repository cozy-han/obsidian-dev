## 구현 규칙

1. **지표**: `strategy/domain/indicator/` 에 계산기 구현
2. **전략**: `strategy/domain/impl/` 에 `TradingStrategy` 구현
3. **팩토리**: `StrategyFactory` 에 타입 등록
4. **시드**: Flyway 마이그레이션으로 기본 설정 추가 (비활성화 상태)
5. **테스트**: 지표 계산 테스트 + 전략 신호 테스트
6. **조합**: 모든 전략은 `CompositeStrategy` 로 AND/OR/가중치 조합 가능
