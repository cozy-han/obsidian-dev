# Claude Code 커스텀 스킬 만들기

## 스킬이란?

`SKILL.md` 파일로 정의하는 슬래시 명령어. Claude의 기능을 확장하는 재사용 가능한 프롬프트 템플릿이다.

## 저장 위치

| 범위 | 경로 |
|------|------|
| 개인용 (모든 프로젝트) | `~/.claude/skills/<스킬명>/SKILL.md` |
| 프로젝트용 | `.claude/skills/<스킬명>/SKILL.md` |

## SKILL.md 기본 구조

```yaml
---
name: my-skill
description: 스킬 설명 (Claude가 자동 호출 판단 기준)
argument-hint: "[인자 힌트]"
---

프롬프트 내용 (마크다운)
$ARGUMENTS 로 사용자 입력을 받음
```

## 주요 프론트매터 옵션

| 필드 | 설명 |
|------|------|
| `name` | 슬래시 명령어명 |
| `description` | Claude 자동 호출 판단 기준 + 자연어 트리거 역할 |
| `disable-model-invocation: true` | 사용자만 `/명령어`로 호출 가능 |
| `user-invocable: false` | Claude만 자동 호출 가능 |
| `context: fork` | 서브에이전트에서 격리 실행 |
| `allowed-tools` | 사용 가능한 도구 제한 |
| `argument-hint` | 자동완성 시 인자 힌트 |

## 호출 방식

| 설정 | `/명령어` | 자연어 자동 호출 |
|------|-----------|----------------|
| 기본값 | O | O |
| `disable-model-invocation: true` | O | X |
| `user-invocable: false` | X | O |

**핵심**: `description`만 잘 작성하면 `/` 없이 자연어로도 스킬이 자동 작동한다.

## 문자열 치환 변수

| 변수 | 설명 |
|------|------|
| `$ARGUMENTS` | 모든 인자 |
| `$ARGUMENTS[N]` / `$N` | N번째 인자 (0부터) |
| `${CLAUDE_SESSION_ID}` | 현재 세션 ID |
| `${CLAUDE_SKILL_DIR}` | SKILL.md가 있는 디렉토리 |

## 동적 컨텍스트 주입

`` !`명령어` `` 문법으로 셸 명령어 결과를 프롬프트에 삽입 가능:

```yaml
변경된 파일: !`gh pr diff --name-only`
```

## 지원 파일 구조

```
my-skill/
├── SKILL.md           # 필수
├── reference.md       # 선택 - 상세 문서
├── examples.md        # 선택 - 사용 예제
└── scripts/
    └── helper.py      # 선택 - 실행 스크립트
```
