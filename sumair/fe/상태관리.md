# 상태관리

Zustand를 사용하며, 대부분 `persist` 미들웨어로 localStorage에 저장. `src/store/` 위치.

## useLoginDataStore
로그인 사용자 데이터 관리 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `loginData` | 사용자 프로필 (ILoginData) |
| `setLoginData()` | 사용자 정보 업데이트 |
| `resetLoginData()` | 로그아웃 (쿠키 삭제 포함) |

## useGuestBookingStore
비회원 예약 조회 정보 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `pnrNo` | 예약 번호 |
| `lastName`, `firstName` | 탑승자 이름 |
| `stDate` | 출발일 (YYYY-MM-DD) |
| `setGuestInfo()` | 정보 설정 |
| `resetGuestInfo()` | 초기화 |

## useAuthNumberStore
인증 플로우 상태 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `authInfo` | `{ clnCd, crtfcDivCd, target }` |
| `setAuthInfo()` | 설정 |
| `resetAuthInfo()` | 초기화 |

## itineraryStore
여정/예약 상세 정보 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `itineraryInfo` | 현재 선택된 항공편 (FlightList) |
| `segmntInfo` | 구간 정보 (SegmntList[]) |
| `PayReceiptRQ` | 결제 영수증 데이터 |
| `advanceInfo` | 사전좌석/서비스 정보 + 가격 |
| `tab` | 활성 탭 (기본: 2) |
| `set*()` / `reset*()` | 각 상태 설정/초기화 |

## webCheckInStore
웹 체크인 플로우 정보 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `checkInfo` | 체크인 항공편 정보 (WebcheckinInfo) |
| `userInfo` | 체크인 대상 탑승자 (UserInfoData[]) |
| `setWebCheckIn()` | 항공편 설정 |
| `setuserInfo()` | 탑승자 설정 |
| `removeAll()` | 전체 초기화 |

## flightStatusStore
운항 현황 (sessionStorage persist - 탭 종료 시 삭제)

| 상태/액션 | 설명 |
|---------|------|
| `flightStatusData` | 출발/도착 스케줄 |
| `confirmData` | 운항 확인 상태 |
| `set*()` / `reset*()` | 설정/초기화 |

## useCountStore
단순 카운터 (localStorage persist)

| 상태/액션 | 설명 |
|---------|------|
| `count` | 카운트 값 |
| `increaseCount()` | 증가 |
| `decreaseCount()` | 감소 |
| `removeAllCount()` | 초기화 |
