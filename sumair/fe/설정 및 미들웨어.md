- **라이브러리:** i18next + react-i18next
- **지원 언어:** 한국어(ko), 영어(en)

## middleware.ts - Edge 미들웨어

인증 상태에 따른 라우트 접근 제어.

### 보호 라우트 (로그인 필요)
로그인하지 않으면 `/customer/main`으로 리다이렉트:
- `/itinerary/myflight` - 내 예약
- `/itinerary/myflight/info` - 예약 상세
- `/mypage/**` - 마이페이지 전체 (`/mypage/setting/withDrawal/Complete` 제외)

### 비로그인 전용 라우트
로그인 상태에서 접근 시 `/`(홈)으로 리다이렉트:
- `/itinerary/myflight/guest` - 비회원 예약 조회
- `/customer/member/**` - 회원가입 (`/customer/member/join/complete` 제외)
- `/customer/main` - 로그인 페이지

### 스마트 리다이렉트
- 로그인 사용자 `/itinerary` 접근 → `/itinerary/myflight`로 자동 이동

## _app.tsx - 앱 설정

### Provider 스택
```
Emotion Cache → MUI Theme → React Query → Localization → History Context
```

### 주요 기능
- 마운트 시 쿠키에서 토큰 자동 로드
- 언어 변경 시 React Query 캐시 무효화
- 라우트 전환 시 글로벌 로딩 스피너
- 500 에러 글로벌 오버레이
- 라우트 변경 시 sessionStorage 자동 정리 (avail/booking 데이터)
- `TokenSetup` 컴포넌트로 네이티브 앱 토큰 브릿지
- Default SEO (SUM Air 메타태그, 구조화 데이터)

### React Query 설정
- 자동 재시도 비활성화
- 글로벌 500 에러 감지
- DevTools (좌하단, 접힌 상태)

## _document.tsx - HTML 문서

- Emotion CSS SSR 삽입 (스타일 깜빡임 방지)
- Pretendard 한국어 폰트 (CDN)
- 파비콘 (다양한 디바이스 크기)
- PWA 메타데이터
- People Events SDK (분석)
- Infobip Live Chat 위젯
