# SUM Air Frontend (Mobile)

## 프로젝트 정보
- **프로젝트명:** sumair-front-mobile
- **경로:** `/Users/han/works/workspace/palnet/sumair/fe/sumair-front-mobile`
- **프레임워크:** Next.js 12 (Pages Router) + TypeScript
- **Node:** >= 20.0.0
- **패키지매니저:** Yarn 1.x

## 개요
섬에어(SUM Air) 항공 모바일 예약 웹사이트. 항공권 검색/예약, 부가서비스, 웹체크인, 마이페이지 등 항공 예약 전 과정을 처리하는 모바일 우선 프론트엔드.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| Framework | Next.js 12 (Pages Router, TypeScript) |
| State | Zustand (localStorage/sessionStorage persist) |
| Server State | React Query (@tanstack/react-query v4) |
| UI | MUI v5 + Emotion CSS-in-JS |
| HTTP | Axios (커스텀 인터셉터) |
| i18n | i18next (한국어/영어) |
| 결제 | Toss Payments (위젯 + BrandPay) |
| 인증 | Cookie 기반 JWT + 토큰 리프레시 큐 |
| 분석 | Amplitude, People Events SDK |
| 채팅 | Infobip Live Chat |
| 날짜 | Moment.js, date-fns |
| 폼 | react-hook-form + yup |
| 캐러셀 | Swiper, react-material-ui-carousel |

## 스크립트
```bash
yarn dev        # 개발 서버
yarn build      # 프로덕션 빌드
yarn build:dev  # 개발환경 빌드 (.env.development 사용)
yarn build:prod # 프로덕션 빌드 (.env.production 사용)
yarn start      # 프로덕션 서버 시작
yarn lint       # ESLint 실행
```
