# 페이지 라우팅

## 메인
| 경로 | 설명 |
|------|------|
| `/` | 메인 랜딩 (항공편 검색, 특가, 추천여행, 날씨) |
| `/contents` | 뉴스/블로그/콘텐츠 |
| `/404` | 404 에러 |
| `/500` | 500 에러 |

## 예약 플로우 (`/booking/`)
순서대로 진행되는 예약 단계:

```mermaid
graph LR
    A[AvailList<br>항공편 선택] --> B[PassengersInfo<br>탑승자 정보]
    B --> C[Ancillary<br>부가서비스]
    C --> D[AdvanceSeat<br>사전좌석]
    D --> E[JourneyCheck<br>예약확인]
    E --> F[AgreeTerms<br>약관동의]
    F --> G[Payment<br>결제]
    G --> H{결과}
    H -->|성공| I[PayComplete]
    H -->|실패| J[PayFail]
    I --> K[Ticket<br>티켓확인]
```

| 경로 | 설명 |
|------|------|
| `/booking/AvailList` | 항공편 가용편 목록/선택 |
| `/booking/PassengersInfo` | 탑승자 정보 입력 |
| `/booking/Ancillary` | 부가서비스 선택 (수하물, 기내식 등) |
| `/booking/AdvanceSeat` | 사전 좌석 배정 |
| `/booking/JourneyCheck` | 예약 내역 확인 |
| `/booking/AgreeTerms` | 약관 동의 |
| `/booking/Payment` | 결제 처리 |
| `/booking/PayComplete` | 결제 성공 |
| `/booking/PayFail` | 결제 실패 |
| `/booking/Ticket` | 예약 확인/티켓 표시 |
