# 채팅 WebSocket (STOMP) 설계

---

## 접속 정보

| 항목 | 값 |
|------|-----|
| 엔드포인트 | `ws://host:8400/ws/chat` |
| 프로토콜 | STOMP over WebSocket |
| 폴백 | SockJS |
| 인증 | STOMP CONNECT 프레임의 `Authorization` 헤더에 JWT 토큰 |

---

## STOMP 브로커 설정

```
Application Prefix:  /app
Broker Prefix:       /topic, /queue
User Prefix:         /user
```

---

## 메시지 흐름

### Client → Server (발행)

| Destination | 설명 | Payload |
|-------------|------|---------|
| `/app/chat/send/{roomId}` | 메시지 전송 | `{ content, msgTypeCd }` |

### Server → Client (구독)

| Destination | 설명 | Payload |
|-------------|------|---------|
| `/topic/room/{roomId}` | 채팅방 메시지 수신 | `ChatWsMessageRs` |
| `/topic/room/{roomId}/event` | 시스템 이벤트 (입장/퇴장/초대) | `ChatWsEventRs` |
| `/user/queue/notification` | 개인 알림 (새 방 초대 등) | `ChatWsEventRs` |

---

## 메시지 포맷

### ChatWsMessageRs (채팅 메시지)
```json
{
  "msgSn": 150,
  "roomId": "ROOM-UUID-0001",
  "senderNo": "USER-UUID-0001",
  "senderNm": "홍길동",
  "content": "안녕하세요",
  "msgTypeCd": "TEXT",
  "crtDt": "2026-03-19T10:30:00"
}
```

### ChatWsEventRs (시스템 이벤트)
```json
{
  "eventType": "INVITE",
  "roomId": "ROOM-UUID-0001",
  "userNo": "USER-UUID-0003",
  "userNm": "이영희",
  "message": "홍길동님이 이영희님을 초대했습니다."
}
```

**eventType 종류:** `JOIN`, `LEAVE`, `INVITE`

---

## 인증 처리 (ChatStompHandler)

STOMP CONNECT 시 `ChannelInterceptor`에서 JWT 검증:

```
1. Client → STOMP CONNECT { headers: { Authorization: "Bearer xxx" } }
2. ChatStompHandler.preSend() 에서:
   - Authorization 헤더에서 JWT 추출
   - TokenProvider.getPayload() 로 검증
   - StompPrincipal 설정 (userNo, userName)
3. 이후 @MessageMapping에서 Principal로 사용자 식별
```

> **주의:** HTTP 필터 체인이 아닌 STOMP ChannelInterceptor에서 인증 처리

---

## 구독 관리

```
1. 클라이언트 접속 시:
   - 내 채팅방 목록 REST API로 조회
   - 각 방의 /topic/room/{roomId} 구독
   - /user/queue/notification 구독 (개인 알림)

2. 새 방 초대 시:
   - /user/queue/notification 으로 초대 알림 수신
   - 해당 방의 /topic/room/{newRoomId} 추가 구독
```

## 확장성 (Phase 1 → Phase 2)

| 항목 | Phase 1 | Phase 2 |
|------|---------|---------|
| 메시지 브로커 | `enableSimpleBroker` (인메모리) | Redis pub/sub 또는 RabbitMQ |
| 인스턴스 | 단일 | 멀티 인스턴스 (Redis 통해 메시지 공유) |
| 입력 중 표시 | 미구현 | `/app/chat/typing/{roomId}` 추가 |
