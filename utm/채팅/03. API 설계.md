# 채팅 REST API 설계

---

## 엔드포인트 목록

### 채팅방 (`ChatRoomController`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| POST | `/api/v1/chat/room/dm` | DM방 생성/조회 | `{ targetUserNo }` | `ChatRoomRs` |
| POST | `/api/v1/chat/room/group` | 그룹방 생성 | `{ roomNm, memberNos[] }` | `ChatRoomRs` |
| GET | `/api/v1/chat/room/slist` | 내 채팅방 목록 | `BaseScrollRq` | `BaseScrollRs<ChatRoomRs>` |
| GET | `/api/v1/chat/room/{roomId}` | 채팅방 상세 | - | `ChatRoomRs` |
| DELETE | `/api/v1/chat/room/{roomId}/leave` | 방 나가기 | - | `void` |

### 메시지 (`ChatMsgController`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| GET | `/api/v1/chat/msg/{roomId}/slist` | 메시지 히스토리 | `BaseScrollRq` | `BaseScrollRs<ChatMsgRs>` |

### 멤버/초대 (`ChatMbrController`)

| Method | Path | 설명 | 요청 | 응답 |
|--------|------|------|------|------|
| GET | `/api/v1/chat/mbr/{roomId}` | 멤버 목록 | - | `List<ChatMbrRs>` |
| POST | `/api/v1/chat/mbr/{roomId}/invite` | 멤버 초대 | `{ userNos[] }` | `void` |
| GET | `/api/v1/chat/user/search` | 유저 검색 | `keyword` | `List<ChatUserSearchRs>` |

---

## 에러 코드

| 코드 | HTTP | 메시지 |
|------|------|--------|
| CHAT4040 | 404 | 채팅방을 찾을 수 없습니다 |
| CHAT4010 | 403 | 채팅방 멤버가 아닙니다 |
| CHAT4000 | 400 | 자기 자신에게 DM을 보낼 수 없습니다 |
| CHAT4041 | 404 | 사용자를 찾을 수 없습니다 |
| CHAT4001 | 400 | 이미 채팅방 멤버입니다 |
| CHAT4002 | 400 | DM 방에는 추가 초대가 불가능합니다 |

## 상세 스펙

### POST `/api/v1/chat/room/dm` — DM방 생성/조회

기존 DM방이 있으면 반환, 없으면 새로 생성.

**Request:**
```json
{
  "targetUserNo": "USER-UUID-0002"
}
```

**Response (ChatRoomRs):**
```json
{
  "roomId": "ROOM-UUID-0001",
  "roomTypeCd": "DM",
  "roomNm": null,
  "lastMsgCn": "안녕하세요",
  "lastMsgDt": "2026-03-19T10:30:00",
  "members": [
    { "userNo": "USER-UUID-0001", "userNm": "홍길동" },
    { "userNo": "USER-UUID-0002", "userNm": "김철수" }
  ]
}
```

### POST `/api/v1/chat/room/group` — 그룹방 생성

**Request (ChatRoomCrtRq):**
```json
{
  "roomNm": "프로젝트 팀",
  "memberNos": ["USER-UUID-0002", "USER-UUID-0003"]
}
```

### GET `/api/v1/chat/msg/{roomId}/slist` — 메시지 히스토리

커서 기반 스크롤 (MSG_SN을 커서로 사용).

**Request:** `?size=20&cursor=150` (MSG_SN 150 이전 메시지 20개)

**Response (BaseScrollRs<ChatMsgRs>):**
```json
{
  "content": [
    {
      "msgSn": 149,
      "senderNo": "USER-UUID-0001",
      "senderNm": "홍길동",
      "msgTypeCd": "TEXT",
      "msgCn": "안녕하세요",
      "crtDt": "2026-03-19T10:30:00"
    }
  ],
  "hasNext": true,
  "cursor": "130"
}
```

### GET `/api/v1/chat/user/search` — 유저 검색 (슬랙 스타일 초대용)

**Request:** `?keyword=김`

**Response:**
```json
[
  { "userNo": "USER-UUID-0002", "userNm": "김철수", "userId": "kim123" },
  { "userNo": "USER-UUID-0005", "userNm": "김영희", "userId": "kimyh" }
]
```
