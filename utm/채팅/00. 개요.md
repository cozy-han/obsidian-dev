# 채팅 모듈 개요

> 작성일: 2026-03-19
> 상태: 설계 완료 → 구현 전
> 프로젝트: KAC-UTM

---

## 배경
KAC-UTM 프로젝트에 채팅 기능 추가. 기존 멀티모듈 Spring Boot 프로젝트 패턴을 따라 구성.

## 1단계 기능 범위

| 카테고리 | 기능 |
|---------|------|
| **인증** | 기존 JWT/OAuth2 활용 (로그인 시 처리) |
| **1:1 채팅 (DM)** | DM 발신, 실시간 메시지 송수신, 이전 메시지 히스토리 |
| **그룹 채팅** | 방 생성, 실시간 메시지 송수신, 멤버 목록 조회, 방 나가기 |
| **초대 (슬랙 스타일)** | 유저 검색, 검색 결과에서 초대, 당 멤버 알림 |

## 2단계 (미포함)
- 읽음 확인
- 파일/이미지 전송
- 푸시 알림
- 입력 중 표시
- 메시지 검색
- 방장 권한/강퇴
