## 환경별 설정 비교

이 프로젝트는 `local`, `dev`, `prod` 3개 프로필을 사용한다.

### 데이터베이스 (MySQL)

| 항목 | Local | Dev | Prod |
|------|-------|-----|------|
| 호스트 | localhost:13306 | AWS RDS (ap-northeast-2) | 10.10.130.11:3306 |
| DB명 | kac-utm | kac-utm | kac-utm |
| HikariCP 풀 | 10 | 40 | 40 |
| Dialect | MySQL8Dialect | MySQL8Dialect | MySQL8Dialect |
| DDL Auto | none | none | none |

> 설정 파일: `common/config/src/main/resources/application-db.yml`

## JWT 보안 설정

각 앱(mngr, web, uss)마다 별도의 secret key를 사용한다.

| 항목 | 값 |
|------|---|
| 알고리즘 | PBES2_HS512_A256KW |
| 인코딩 | A256GCM |
| Access Token TTL | 7,200초 (2시간) |
| Refresh Token TTL | 86,400초 (24시간) |
| 로그인 실패 제한 | 5회 |

> 설정 파일: `common/config/src/main/resources/application-security.yml`

## Docker Compose 상세

```yaml
services:
  redis:
    image: redis:alpine
    ports: 16379:6379
    password: palnet!234

  db (MySQL):
    image: mysql
    ports: 13306:3306
    database: kac-utm
    user: kac / palnet!234

  rabbitmq:
    image: rabbitmq:4-management
    ports:
      - 35672:5672 (AMQP)
      - 45672:15672 (관리 콘솔)
      - 31883:1883 (MQTT)
    plugins: mqtt, shovel, shovel_management
```
