## 1. DM 메시지 전송

```
Client                          Server (app/chat)                 DB
  │                                │                               │
  ├─ STOMP CONNECT (JWT) ─────────►│                               │
  │                                ├─ JWT 검증 (ChatStompHandler)  │
  │◄─ CONNECTED ──────────────────┤                               │
  │                                │                               │
  ├─ SUBSCRIBE /topic/room/{id} ──►│                               │
  │                                │                               │
  ├─ SEND /app/chat/send/{id} ───►│                               │
  │   { content: "안녕하세요" }      ├─ 멤버십 검증 ──────────────────►│
  │                                ├─ 메시지 저장 (CHAT_MSG_BSC) ──►│
  │                                ├─ LAST_MSG 업데이트 ────────────►│
  │                                │                               │
  │◄─ MESSAGE /topic/room/{id} ───┤ (브로드캐스트)                   │
  │   { msgSn, sender, content }   │                               │
```

## 2. DM방 생성

```
1. POST /api/v1/chat/room/dm { targetUserNo: "B" }

2. ChatRoomService.createDmRoom():
   a. QueryDSL로 A↔B 간 기존 DM방 검색
      - ROOM_TYPE_CD = 'DM'
      - 두 유저 모두 활성 멤버 (DEL_YN = 'N')
   b. 존재하면 → 기존 방 반환
   c. 없으면:
      - CHAT_ROOM_BSC 생성 (roomTypeCd='DM', roomNm=null)
      - CHAT_ROOM_MBR × 2 (A=OWNER, B=MEMBER)
      - 새 방 반환

3. Response: ChatRoomRs { roomId, members, ... }
```

## 3. 그룹방 생성

```
1. POST /api/v1/chat/room/group { roomNm: "팀채팅", memberNos: ["B", "C"] }

2. ChatRoomService.createGroupRoom():
   a. CHAT_ROOM_BSC 생성 (roomTypeCd='GROUP', roomNm='팀채팅')
   b. CHAT_ROOM_MBR: 생성자 A = OWNER
   c. CHAT_ROOM_MBR: B, C = MEMBER
   d. SYSTEM 메시지: "A님이 방을 만들었습니다."
   e. SYSTEM 메시지: "B님, C님이 초대되었습니다."
   f. /user/{B}/queue/notification → 초대 알림
   g. /user/{C}/queue/notification → 초대 알림

3. Response: ChatRoomRs
```

## 4. 슬랙 스타일 초대

```
1. GET /api/v1/chat/user/search?keyword=김
   → COM_USER_BSC에서 검색
   → List<ChatUserSearchRs> 반환

2. POST /api/v1/chat/mbr/{roomId}/invite { userNos: ["D", "E"] }

3. ChatMbrService.inviteMembers():
   a. 요청자가 방 멤버인지 검증
   b. DM방이면 → 에러 (CHAT4002)
   c. 이미 활성 멤버면 → 스킵
   d. CHAT_ROOM_MBR 추가 (mbrRoleCd='MEMBER')
   e. SYSTEM 메시지: "A님이 D님, E님을 초대했습니다."
   f. /topic/room/{roomId}/event → 이벤트 브로드캐스트
   g. /user/{D}/queue/notification → 개인 알림
   h. /user/{E}/queue/notification → 개인 알림
```

## 5. 방 나가기

```
1. DELETE /api/v1/chat/room/{roomId}/leave

2. ChatRoomService.leaveRoom():
   a. DM방:
      - CHAT_ROOM_MBR.DEL_YN = 'Y' (숨김 처리)
      - 방 자체는 유지 (상대방이 다시 DM 시 복원)
   b. GROUP방:
      - CHAT_ROOM_MBR.DEL_YN = 'Y'
      - SYSTEM 메시지: "A님이 나갔습니다."
      - /topic/room/{roomId}/event → LEAVE 이벤트
      - 남은 활성 멤버 0명 → CHAT_ROOM_BSC.DEL_YN = 'Y'
```

## 6. 메시지 히스토리 조회

```
1. GET /api/v1/chat/msg/{roomId}/slist?size=20&cursor=150

2. ChatMsgService.getMessages():
   a. 멤버십 검증
   b. QueryDSL: WHERE ROOM_ID = ? AND MSG_SN < 150
                ORDER BY MSG_SN DESC
                LIMIT 21 (size + 1)
   c. 21개 조회되면 hasNext = true, 마지막 1개 제거
   d. 결과를 역순으로 반환 (오래된 → 최신)

3. Response: BaseScrollRs<ChatMsgRs> { content, hasNext, cursor }
```
