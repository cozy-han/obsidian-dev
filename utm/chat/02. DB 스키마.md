# 채팅 DB 스키마

---

## 테이블 목록

| 테이블 | 설명 |
|--------|------|
| `CHAT_ROOM_BSC` | 채팅방 기본 |
| `CHAT_ROOM_MBR` | 채팅방 멤버 |
| `CHAT_MSG_BSC` | 채팅 메시지 |

## ERD

```
CHAT_ROOM_BSC (1) ──── (N) CHAT_ROOM_MBR
      │                         │
      │                         │ USER_NO → COM_USER_BSC (기존 유저 테이블)
      │
      └── (1) ──── (N) CHAT_MSG_BSC
```

## CHAT_ROOM_BSC (채팅방 기본)

```sql
CREATE TABLE CHAT_ROOM_BSC (
    ROOM_ID         VARCHAR(36)     NOT NULL    COMMENT '채팅방ID (UUID)',
    SITE_ID         VARCHAR(20)     NOT NULL    COMMENT '사이트ID',
    ROOM_TYPE_CD    VARCHAR(10)     NOT NULL    COMMENT '채팅방유형코드 (DM/GROUP)',
    ROOM_NM         VARCHAR(200)    NULL        COMMENT '채팅방명 (GROUP only)',
    LAST_MSG_CN     VARCHAR(4000)   NULL        COMMENT '마지막메시지내용',
    LAST_MSG_DT     DATETIME        NULL        COMMENT '마지막메시지일시',
    USE_YN          CHAR(1)         NOT NULL    DEFAULT 'Y' COMMENT '사용여부',
    DEL_YN          CHAR(1)         NOT NULL    DEFAULT 'N' COMMENT '삭제여부',
    CRT_DT          DATETIME        NOT NULL    COMMENT '생성일시',
    CREATR_NO       VARCHAR(36)     NOT NULL    COMMENT '생성자번호',
    MDFCN_DT        DATETIME        NOT NULL    COMMENT '수정일시',
    MDFR_NO         VARCHAR(36)     NOT NULL    COMMENT '수정자번호',
    PRIMARY KEY (ROOM_ID),
    INDEX IDX_CHAT_ROOM_BSC_01 (SITE_ID, DEL_YN),
    INDEX IDX_CHAT_ROOM_BSC_02 (LAST_MSG_DT)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='채팅방 기본';
```

## CHAT_ROOM_MBR (채팅방 멤버)

```sql
CREATE TABLE CHAT_ROOM_MBR (
    MBR_SN          BIGINT          NOT NULL    AUTO_INCREMENT COMMENT '멤버일련번호',
    ROOM_ID         VARCHAR(36)     NOT NULL    COMMENT '채팅방ID',
    USER_NO         VARCHAR(36)     NOT NULL    COMMENT '사용자번호',
    MBR_ROLE_CD     VARCHAR(10)     NOT NULL    DEFAULT 'MEMBER' COMMENT '멤버역할코드 (OWNER/MEMBER)',
    JOIN_DT         DATETIME        NOT NULL    COMMENT '참여일시',
    LAST_READ_MSG_SN BIGINT         NULL        COMMENT '마지막읽은메시지SN (2단계용)',
    DEL_YN          CHAR(1)         NOT NULL    DEFAULT 'N' COMMENT '삭제여부 (나가기=Y)',
    CRT_DT          DATETIME        NOT NULL    COMMENT '생성일시',
    CREATR_NO       VARCHAR(36)     NOT NULL    COMMENT '생성자번호',
    MDFCN_DT        DATETIME        NOT NULL    COMMENT '수정일시',
    MDFR_NO         VARCHAR(36)     NOT NULL    COMMENT '수정자번호',
    PRIMARY KEY (MBR_SN),
    UNIQUE INDEX UDX_CHAT_ROOM_MBR_01 (ROOM_ID, USER_NO),
    INDEX IDX_CHAT_ROOM_MBR_01 (USER_NO, DEL_YN),
    CONSTRAINT FK_CHAT_ROOM_MBR_01 FOREIGN KEY (ROOM_ID) REFERENCES CHAT_ROOM_BSC (ROOM_ID)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='채팅방 멤버';
```

## CHAT_MSG_BSC (채팅 메시지)

```sql
CREATE TABLE CHAT_MSG_BSC (
    MSG_SN          BIGINT          NOT NULL    AUTO_INCREMENT COMMENT '메시지일련번호',
    ROOM_ID         VARCHAR(36)     NOT NULL    COMMENT '채팅방ID',
    SNDR_NO         VARCHAR(36)     NOT NULL    COMMENT '발신자번호',
    MSG_TYPE_CD     VARCHAR(10)     NOT NULL    DEFAULT 'TEXT' COMMENT '메시지유형코드 (TEXT/SYSTEM)',
    MSG_CN          TEXT            NOT NULL    COMMENT '메시지내용',
    DEL_YN          CHAR(1)         NOT NULL    DEFAULT 'N' COMMENT '삭제여부',
    CRT_DT          DATETIME        NOT NULL    COMMENT '생성일시',
    CREATR_NO       VARCHAR(36)     NOT NULL    COMMENT '생성자번호',
    MDFCN_DT        DATETIME        NOT NULL    COMMENT '수정일시',
    MDFR_NO         VARCHAR(36)     NOT NULL    COMMENT '수정자번호',
    PRIMARY KEY (MSG_SN),
    INDEX IDX_CHAT_MSG_BSC_01 (ROOM_ID, MSG_SN),
    INDEX IDX_CHAT_MSG_BSC_02 (ROOM_ID, CRT_DT),
    CONSTRAINT FK_CHAT_MSG_BSC_01 FOREIGN KEY (ROOM_ID) REFERENCES CHAT_ROOM_BSC (ROOM_ID)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='채팅 메시지';
```

## 설계 포인트

| 항목                   | 설명                                                          |
| -------------------- | ----------------------------------------------------------- |
| **MSG_SN 커서**        | BIGINT AUTO_INCREMENT → 삽입 순서 보장, 커서 기반 페이지네이션에 사용          |
| **LAST_MSG 비정규화**    | 방 목록 조회 시 메시지 테이블 JOIN 없이 미리보기 제공                           |
| **LAST_READ_MSG_SN** | 2단계 읽음 확인 대비, Phase 1에서는 NULL 허용                            |
| **DM 방 이름**          | `ROOM_TYPE_CD = 'DM'`일 때 `ROOM_NM`은 NULL, 클라이언트에서 상대방 이름 표시 |
| **소프트 삭제**           | `DEL_YN = 'Y'`로 논리 삭제, 물리 삭제하지 않음                           |
