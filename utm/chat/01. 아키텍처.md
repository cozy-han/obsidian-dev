# 채팅 모듈 아키텍처

---

## 핵심 결정: 별도 앱 모듈 (`app/chat`) 신설

| 항목 | 결정 | 이유 |
|------|------|------|
| 앱 분리 | `app/chat` 별도 Spring Boot 앱 | 기존 `app/ws`는 드론 제어용 WebFlux, 도메인이 다름 |
| WebSocket | STOMP over WebSocket (서블릿 스택) | pub/sub 시맨틱이 채팅방 구조에 적합 |
| 포트 | **8400** | 기존 web(8080), ws(8300)과 충돌 방지 |
| 인증 | 기존 JWT 재사용 | `common/security`의 `TokenProvider` 활용 |

## 기존 인프라 재사용

| 모듈 | 활용 내용 |
|------|----------|
| `common/security` | JWT 검증, `SecurityHelper`, `TokenProvider` |
| `common/models` | `BaseScrollRq/Rs` (커서 기반 페이지네이션) |
| `common/exception` | `BaseException`, `ErrorCode` |
| `common/redis` | 세션 관리, 향후 멀티 인스턴스 pub/sub |
| `common/utils` | `JsonUtil` |
| `module/common` | `ComUserBscEntity` (유저 검색) |

## 모듈 구조

### `module/chat/` (도메인 모듈)
```
module/chat/src/main/java/com/kac/utm/chat/
├── jpa/
│   ├── entity/
│   │   ├── ChatRoomBscEntity.java      # 채팅방 기본
│   │   ├── ChatRoomMbrEntity.java      # 채팅방 멤버
│   │   └── ChatMsgBscEntity.java       # 채팅 메시지
│   └── repository/
│       ├── ChatRoomBscRepository.java
│       ├── ChatRoomBscBaseRepository.java (QueryDSL)
│       ├── ChatRoomMbrRepository.java
│       ├── ChatMsgBscRepository.java
│       └── ...Impl.java
├── model/
│   ├── ChatRoomRs.java                 # 방 응답
│   ├── ChatRoomCrtRq.java              # 방 생성 요청
│   ├── ChatMsgRs.java                  # 메시지 응답
│   ├── ChatMbrRs.java                  # 멤버 응답
│   ├── ChatUserSearchRs.java           # 유저 검색 응답
│   ├── ChatInviteRq.java               # 초대 요청
│   └── constant/
│       ├── ChatRoomTypeCd.java          # DM / GROUP
│       ├── ChatMbrRoleCd.java           # OWNER / MEMBER
│       └── ChatMsgTypeCd.java           # TEXT / SYSTEM
└── service/
    ├── ChatRoomService.java
    ├── ChatMsgService.java
    └── ChatMbrService.java
```

### `app/chat/` (애플리케이션 모듈)
```
app/chat/src/main/java/com/kac/utm/chat/
├── ChatApplication.java
├── config/
│   ├── ChatJpaConfig.java
│   ├── ChatQuerydslConfig.java
│   ├── ChatWebSocketConfig.java         # STOMP 설정
│   ├── ChatSecurityConfig.java
│   └── ChatRedisConfig.java
├── api/
│   ├── ChatRoomController.java
│   ├── ChatMsgController.java
│   └── ChatMbrController.java
└── ws/

    └── ChatWsController.java            # @MessageMapping 핸들러
```
